## Usage

Use the `saturate-{amount}` utilities alongside the `backdrop-filter` utility to control an element's backdrop saturation.

```html rose
<template preview>
  <div class="h-full w-full p-4 rounded-2xl bg-rose-200">
    <div class="flex justify-around mb-8">
      <div class="flex-1">
        <p class="text-center text-sm text-rose-800 font-medium mb-1">.backdrop-saturate-0</p>
        <div class="mx-auto h-24 w-24 rounded-xl shadow-md backdrop-filter backdrop-saturate-0">
        </div>
      </div>
      <div class="flex-1">
        <p class="text-center text-sm text-rose-800 font-medium mb-1">.backdrop-saturate-50</p>
        <div class="mx-auto h-24 w-24 rounded-xl shadow-md backdrop-filter backdrop-saturate-50">
        </div>
      </div>
      <div class="flex-1">
        <p class="text-center text-sm text-rose-800 font-medium mb-1">.backdrop-saturate-100</p>
        <div class="mx-auto h-24 w-24 rounded-xl shadow-md backdrop-filter backdrop-saturate-100">
        </div>
      </div>
    </div>
    <div class="flex justify-around">
      <div class="flex-1">
        <p class="text-center text-sm text-rose-800 font-medium mb-1">.backdrop-saturate-150</p>
        <div class="mx-auto h-24 w-24 rounded-xl shadow-md backdrop-filter backdrop-saturate-150">
        </div>

```

## Responsive

To control an element's backrop saturation at a specific breakpoint, add a `{screen}:` prefix to any existing backdrop saturate utility. For example, use `md:backdrop-saturate-150` to apply the `backdrop-saturate-150` utility at only medium screen sizes and above.

```html
<div class="backdrop-filter backdrop-saturate-110 **md:backdrop-saturate-150** ...">
  <!-- ... -->
</div>
```

For more information about Tailwind's responsive design features, check out the [Responsive Design](/docs/responsive-design) documentation.
