## Usage

Use the `saturate-{amount}` utilities alongside the `filter` utility to control an element's saturation.

```html rose
<template preview>
  <div class="flex justify-around mb-8">
    <div class="flex-1">
      <p class="text-center text-sm text-rose-600 font-medium mb-1">.saturate-0</p>
      <div class="mx-auto h-24 w-24 rounded-xl shadow-md bg-rose-400 filter saturate-0">
      </div>
    </div>
    <div class="flex-1">
      <p class="text-center text-sm text-rose-600 font-medium mb-1">.saturate-50</p>
      <div class="mx-auto h-24 w-24 rounded-xl shadow-md bg-rose-400 filter saturate-50">
      </div>
    </div>
    <div class="flex-1">
      <p class="text-center text-sm text-rose-600 font-medium mb-1">.saturate-100</p>
      <div class="mx-auto h-24 w-24 rounded-xl shadow-md bg-rose-400 filter saturate-100">
      </div>
    </div>
  </div>
  <div class="flex justify-around">
    <div class="flex-1">
      <p class="text-center text-sm text-rose-600 font-medium mb-1">.saturate-150</p>
      <div class="mx-auto h-24 w-24 rounded-xl shadow-md bg-rose-400 filter saturate-150">
      </div>
    </div>
    <div class="flex-1">
      <p class="text-center text-sm text-rose-600 font-medium mb-1">.saturate-200</p>
      <div class="mx-auto h-24 w-24 rounded-xl shadow-md bg-rose-400 filter saturate-200">
      </div>
    </div>
    <div class="flex-1"></div>
  </div>
</template>

<div class="filter **saturate-0** ..."></div>
<div class="filter **saturate-50** ..."></div>
<div class="filter **saturate-100** ..."></div>
<div class="filter **saturate-150** ..."></div>
<div class="filter **saturate-200** ..."></div>
```

## Responsive

To control an element's saturation at a specific breakpoint, add a `{screen}:` prefix to any existing saturate utility. For example, use `md:saturate-150` to apply the `saturate-150` utility at only medium screen sizes and above.

```html
<div class="filter saturate-110 **md:saturate-150** ...">
  <!-- ... -->
</div>
```

For more information about Tailwind's responsive design features, check out the [Responsive Design](/docs/responsive-design) documentation.
